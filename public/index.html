Quick Start

npm install
export PUBLORA_API_KEY=your_key
npm start

Server runs at http://localhost:3000

REST API

Create a post:

curl -X POST "http://localhost:3000/api/v1/create-post" \
  -H "x-publora-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "Hello!", "platforms": ["linkedin-xxx"], "scheduledTime": "2026-06-05T10:00:00Z"}'

List posts:

curl "http://localhost:3000/api/v1/posts?status=scheduled" \
  -H "x-publora-key: YOUR_API_KEY"

MCP Server

Configure in Claude Desktop:

{
  "mcpServers": {
    "publora-mvp": {
      "type": "http",
      "url": "http://localhost:3000/mcp",
      "headers": { "Authorization": "Bearer YOUR_API_KEY" }
    }
  }
}

Available tools: list_posts, create_post, get_post, update_post, delete_post, list_connections, linkedin_post_stats, linkedin_create_comment
